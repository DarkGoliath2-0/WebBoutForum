Pour installer le forum php sur votre site web suivez c'est instruction.

- 1 => decompresser l'archive à l'aide des 3 petit points : unarchive
- 2 => Créer un redirection depuis votre site web vers : /phpBB3/
- 3 =>  Suivez la procédure d'installation dans l'interface..
- 4 => Supprimé ou renommer le fichier "install" : /phpBB3/install
- 5 => L'installation est terminé, il suffit de cliquer sur votre lien pour accéder au forum..
